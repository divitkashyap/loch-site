# loch-site

Marketing/landing page for [Loch](https://github.com/divitkashyap/loch-app) — static HTML/CSS/JS, no build step, deployed on Vercel.

- `index.html` / `styles.css` / `script.js` — the page
- `assets/` — screenshots, logo, favicon, OG image (sourced from the app's own release screenshots)

Edit and push to `main`; Vercel redeploys automatically.
