Quiet by default

It shows up when something needs you.
Otherwise, it disappears.

A permission prompt. A finished build. A song change. That's when Loch takes the notch — every surface has an attention-only mode, so a running process never just sits there taking up space.

Install

One line, one download.

$ brew install --cask divitkashyap/tap/loch

Prefer not to use Homebrew? Grab the zip from GitHub Releases and drop it in /Applications.

macOS 14 Sonoma or later. Unsigned for now — clear Gatekeeper once after verifying the checksum, see the install guide. Free. No account. No telemetry.
